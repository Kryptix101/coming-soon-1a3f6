# Coming Soon Page

A beautiful, minimal coming-soon page built with Next.js 14 and Tailwind CSS. Features a dark theme with cyan accents, smooth animations, and an email capture form.

## Features

- 🌙 Dark theme with cyan accent colors
- 📱 Fully responsive design
- ✨ Smooth animations and transitions
- 📧 Email capture form with validation
- 🎨 Modern gradient backgrounds and glass-morphism effects
- ⚡ Built with Next.js 14 App Router
- 🎯 TypeScript for type safety

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Customization

### Colors
The accent color can be changed by modifying the cyan color values in `tailwind.config.js`.

### Content
Update the title, subtitle, and other content in `app/page.tsx`.

### Email Integration
Replace the simulated email submission in the `handleSubmit` function with your preferred email service (Mailchimp, ConvertKit, etc.).

## Deploy

The easiest way to deploy is using [Vercel](https://vercel.com):

```bash
npm run build
```

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- Modern CSS animations
